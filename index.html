📚 À propos de FileManager

FileManager est une application moderne de gestion de fichiers en ligne, conçue pour offrir une expérience utilisateur exceptionnelle.

✨ Caractéristiques principales :

  • Interface moderne avec design 3D
  • Glisser-déposer intuitif
  • Prévisualisation des fichiers
  • Stockage sécurisé avec Firebase
  • Design responsive

Version 1.0 - Développé avec passion ❤️

🚀 Fonctionnalités

📁 Upload avancé

Glissez-déposez vos fichiers ou utilisez le sélecteur. Support multi-fichiers.

👁️ Prévisualisation

Visualisez directement vos images, PDFs et obtenez des infos sur vos fichiers.

📱 Responsive

Interface adaptée à tous les appareils : mobile, tablette, desktop.

🔒 Sécurité

Vos fichiers sont stockés de manière sécurisée avec Firebase.

❓ Aide & Support

📝 Comment utiliser FileManager ?

  1. Upload : Glissez vos fichiers dans la zone ou cliquez "Choisir des fichiers"
  2. Visualiser : Cliquez sur "👁️ Afficher" pour prévisualiser
  3. Télécharger : Utilisez le bouton "⬇️ Télécharger"
  4. Supprimer : Cliquez sur "🗑️ Supprimer" (avec confirmation)
